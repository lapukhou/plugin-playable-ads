# Assets Directory

Custom per-project assets for playable ads go here (sprites, inline SVG buttons, base64-encoded sounds).

Base game templates are **not** stored here — they are generated by `scripts/generate_playable_template.py` (`match3`, `runner`, `bubble`), which emits a complete single-file HTML playable.

Keep anything you add inline-friendly (SVG or small base64 images) so the final ad stays a single offline file under the network size limit.
